# Cosmic Landing Template

Premium dark landing page with 12 motion primitives. Next.js 16 + Framer Motion + Tailwind.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/frankxai/cosmic-landing-template)

## Motion Primitives Included

| Primitive | What It Does |
|-----------|-------------|
| **LiquidGlass** | 4-layer glass with noise, sheen, tilt, outer glow |
| **SplitText** | Character-by-character blur-to-focus reveal |
| **TiltCard** | 3D perspective tilt on hover via spring physics |
| **GlowCard** | Cursor-following border glow |
| **Magnetic** | Cursor attraction for CTAs |
| **Reveal** | Scroll-triggered blur-to-focus |
| **GradientMesh** | Animated radial gradient background |
| **NumberTicker** | Count-up animation with spring |
| **Marquee** | Infinite horizontal scroll |
| **AnimatedBeam** | SVG path between DOM refs |
| **ShimmerButton** | CSS shine sweep |
| **CursorFollower** | Custom dot+ring cursor |

## Quick Start

```bash
git clone https://github.com/frankxai/cosmic-landing-template.git
cd cosmic-landing-template
pnpm install
pnpm dev
```

No API keys needed. No database. Just a landing page.

## Stack

- Next.js 16 + React 19
- Framer Motion 11
- Tailwind CSS
- TypeScript (strict)

## Design Tokens

- Background: `#09090b`
- Primary: `#00bcd4` (teal)
- Accent: `#ffd700` (gold)
- Fonts: Space Grotesk (display), Inter (body), JetBrains Mono (code)
